
Neon Button Hover Effects

This project demonstrates neon-style button hover effects using HTML and CSS.
It includes visually attractive buttons with glowing effects inspired by popular brands like Coca-Cola, Pepsi, and Sprite.

The main goal of this project is to showcase modern UI button designs that can be used in websites, landing pages, or frontend practice projects.
